# Видеозвонок

Приложение для видеозвонков в стиле FaceTime. Работает в браузере — без установки программ.

## Возможности

- Видео и аудио звонки 1-на-1
- Демонстрация экрана
- Рисование поверх демонстрации экрана (оба участника)
- Работа через интернет (TURN-серверы для обхода NAT)

## Локальный запуск

```bash
cd video-call
pip3 install -r requirements.txt
python3 server.py
```

Откройте http://localhost:3000

## Деплой на Render (бесплатно)

1. Загрузите репозиторий на GitHub
2. Зайдите на [render.com](https://render.com) → New → Blueprint
3. Подключите репозиторий — Render подхватит `render.yaml`
4. После деплоя откройте URL вида `https://video-call-xxxx.onrender.com`

## Как пользоваться

1. Нажмите **«Создать новую комнату»**
2. Отправьте ссылку другу
3. Друг открывает ссылку и нажимает **«Присоединиться»**
4. Разрешите доступ к камере и микрофону

### Демонстрация экрана и рисование

- Нажмите кнопку **монитора** в панели управления
- Выберите экран или окно для демонстрации
- Появится панель рисования: карандаш, цвет, толщина, очистка
- Оба участника могут рисовать поверх демонстрации

## Технологии

- WebRTC — видео/аудио P2P
- WebSocket — сигнализация и синхронизация рисунков
- Python (aiohttp) — сервер
